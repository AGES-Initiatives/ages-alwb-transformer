Preface

These services are provided by AGES Initiatives, Inc. and were generated using AGES Liturgical Workbench.

They are provided as an ePub file so that you can view them offline (when you do not have Internet access) and so that you can easily make the font larger.

Before each service, there is an outline. There are two uses of the outlines. First, it allows you to learn the structure of the service. Second, it allows you to quickly jump to the page where a certain part of the service occurs. The outline includes the first few words of hymns. This way, if you arrive after a service has started, you can more easily find the point in the service to view on the reader.

Please note that when viewing bilingual services, if you make the font very large, some words might not fit on a line. In such cases, the ending of the word will wrap around to the next line. If your eReader doesn't know how to hyphenate a particular language (e.g. Greek), it will break the word, but without a hyphen mark. If you do not want this, simply make the font smaller. Also, some eReaders might justify the text. If you do not want the text to be justified, look for a setting in your reader that allows you to turn it off.
